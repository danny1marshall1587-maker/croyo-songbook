# APP STATE SUMMARY & DYSLEXA-SONGBOOK MISSION

## 1. PROJECT OVERVIEW
**Current Project Name:** Cryo Songbook (Fork of OpenSong Tablet)
**Future Project Name:** DyslexaSongbook
**Mission:** To create the world's most accessible, high-performance digital songbook for musicians, with a focus on dyslexia-friendly typography and AI-driven automation.

---

## 2. KEY FEATURES ADDED (CRYO-VERSION)
The app has evolved significantly beyond the standard OpenSong Tablet:

*   **AI Cryo-Follow:** 
    *   Integrates **Gemma AI** with **Vosk ASR**.
    *   Real-time listening that aligns lyrics/scrolling with the singer's voice.
    *   Bridge logic in `MainActivity` and `AiAgentManager`.
*   **Dyslexia-First Accessibility:**
    *   Custom font loader for **OpenDyslexic** and other user-provided accessibility fonts.
    *   High-contrast theme engine in `CryoSettingsFragment`.
*   **Cryo-Flow Visuals:**
    *   A dynamic micro-animation engine for the prompter.
    *   Patterns like "Tide" provide a subconscious rhythmic reference without visual clutter.
*   **Bluetooth Shield Service:**
    *   `BluetoothShieldService.kt` runs in the background to ensure Bluetooth pedals (like Lekato WT-3) remain connected despite Android's aggressive battery management.
*   **Face-Gesture Navigation:**
    *   **MediaPipe Face Landmarker** integration.
    *   Hands-free page turning via facial movements/gestures.
*   **Storage Hardening:**
    *   Custom SAF (Storage Access Framework) handling for Android 11+ ensuring "All Files Access" where possible to prevent permission-related crashes.
